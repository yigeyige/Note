#*逻辑运算符*

1. 逻辑与 &  
    ```
    int i = 10;
    int j = 20;
   
    System.out.println((i++ > 100) & (j++ > 100)); 
    //输出false
    System.out.println(i);
    //输出11
    System.out.println(j);
    //输出21
    ```
2. 短路与 &&  
    ```
    int i = 10;
    int j = 20;
       
    System.out.println((i++ > 100) & (j++ > 100)); 
    //输出false
    System.out.println(i);
    //输出11
    System.out.println(j);
    //输出20
    ```  
    
    备注：逻辑&和短路&&的区别就是 短路&&计算到第一个false之后就不会计算后续的逻辑运算
